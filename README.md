# 刺猬搬搬官网首版

这是一个无需构建步骤的静态官网首版，适合先快速确认品牌方向、项目定位和页面结构。

## 文件说明

- `index.html`：页面结构与文案
- `styles.css`：视觉样式
- `script.js`：轻量滚动显现动画

## 本地打开方式

直接双击 `index.html`，或在终端里执行：

```bash
open index.html
```

## 建议优先替换的内容

1. 首页主标题和副标题
2. 联系方式（邮箱 / 微信 / 地址）
3. “为什么是我们”中的具体资源与能力表述
4. 后续可以补一页：真实案例 / 数据成果 / 合作伙伴
